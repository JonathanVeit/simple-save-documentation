Detailed Description

Member with this attribute can have their references be saved and loaded. GameObject or Component Use this property to save and load references of type .

The type of the member has to be GameObject, Component or any type derived from Component. The member has to be saved by a SaveItem or marked as a SaveVar in order to save the reference.

public class ExampleClass : MonoBehaviour
{
// the references of these fields will be saved and loaded together with the component
[SerializeField] [SaveRef] GameObject gameObjectField;
[SerializeField] [SaveRef] Transform transformField;
}
public static class MyGlobalVariables
{
// the references of this field will be saved and loaded
[SaveVar] [SaveRef] static GameObject gameObject;
}
