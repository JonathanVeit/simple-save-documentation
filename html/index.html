Simple Save
An easy to use framework to implement saving and loading features in unity projects.
Simple Save

This is the documentation of Simple Save. The framework is still in development and may contain bugs.
Feel free to contact me if you run into problems: https://jonathanveit.wordpress.com/contact/


Installation

Step 1: Download the package.

  • You can download the package using the PackageManager and the following git link: https://github.com/JonathanVeit/simple-save-package.git
  • Install the package using this tutorial: https://docs.unity3d.com/Manual/upm-ui-giturl.html
  • The package should be installed and ready to use!

Step 2: Setup the unity project.

  • Please change the Api Compatibility Level to .NEt 4.x Equivalent: https://docs.microsoft.com/de-de/visualstudio/gamedev/unity/unity-scripting-upgrade
  • Simple Save will create some ScriptableObject instance under "Assets/Simple Save/". Please do not touch any of them.
  • The following ScriptableObject instances can be added to your .gitignore file:
    Assets/Simple Save/Resources/Container/SaveVarInfoContainer.asset
    Assets/Simple Save/Resources/Container/ReferenceInfoContainer.asset


Known Issues

Duplicated Newtonsoft.Json.dll

In case you already have a version of Newtonsoft.Json in your project, please delete the following folder in the Simple Save package:
Package/Simple Save/Scripts/Plugins/Assemblies

Empty settings.

There is a bug resulting in empty default settings after installing the package. In case this happens, please delete the the following file:
Assets/Simple Save/Resources/Container/SettingsContainer.asset
